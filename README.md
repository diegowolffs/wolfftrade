# WolffTrade is a backend-focused trading platform inspired by real-world stock exchanges like Nasdaq.
The project is built to study and apply backend engineering best practices using .NET, Clean Architecture, SOLID, and Domain-Driven Design (DDD).

The platform follows a microservices architecture, where services communicate asynchronously using RabbitMQ, promoting loose coupling, scalability, and resilience.

This repository aims to simulate core trading scenarios such as:

Buy and sell order creation

Order validation and lifecycle management

Event-driven communication between services

Clear domain modeling and business rules

The main goal of WolffTrade is learning through practice, focusing on high-quality backend design, real-world patterns, and production-ready code structure.

WolffTrade é uma plataforma de negociação focada em backend, inspirada em bolsas de valores reais como a Nasdaq.
O projeto foi criado para estudar e aplicar boas práticas de engenharia de software backend utilizando .NET, Clean Architecture, SOLID e Domain-Driven Design (DDD).

A plataforma segue uma arquitetura de microserviços, onde os serviços se comunicam de forma assíncrona através do RabbitMQ, garantindo baixo acoplamento, escalabilidade e resiliência.

Este repositório simula cenários centrais de negociação, como:

Criação de ordens de compra e venda

Validação e ciclo de vida das ordens

Comunicação orientada a eventos entre serviços

Modelagem de domínio clara e regras de negócio bem definidas

O principal objetivo do WolffTrade é aprendizado prático, com foco em design de backend de alta qualidade, padrões reais de mercado e código preparado para ambientes produtivos.
